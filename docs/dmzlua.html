DMZ Lua API Reference Guide

dmz.config

The dmz.config class is a wrapper for the dmz::Config class.

dmz.config.get_global ()

Returns a dmz.config object containing the global config data.

dmz.config.new ([name])

Returns a new dmz.config object. The name is used to set the new dmz.config objects tag.

dmz.config.is_a (value)

Returns a dmz.config if value is a dmz.config object. Returns nil if it is not.

dmz.config:get_name ()

Returns the name of the dmz.config object. Returns nil if the dmz.config object is empty.

dmz.config:get_attributes ()

Returns a table containing the attributes stored in the dmz.config object. Returns nil if the dmz.config object is empty.

dmz.config:store_attribute (name, value)

Store an attribute of name with value in the dmz.config object. Returns true if the attribute was successfully stored.

dmz.config:lookup_attribute (name)

Returns the value of the attribute name. Returns nil if the dmz.config object does not contain an attribute with the given name.

dmz.config:lookup_boolean (name, [default])

Looks up the attribute value name and returns it as a boolean. If the attribute name is not found, then default is returned.

dmz.config:lookup_string (name, [default])

Looks up the attribute value name and returns it as a string. If the attribute name is not found, then default is returned.

dmz.config:lookup_number (name, [default])

Looks up the attribute value name and returns it as a number. If the attribute name is not found, then default is returned.

dmz.config:lookup_handle (name, [default])

Looks up the attribute value name and returns it as a dmz.handle. If the attribute name is not found, then default is returned.

dmz.config:lookup_matrix (name, [default])

Looks up the attribute value name and returns it as a dmz.matrix. If the attribute name is not found, then default is returned.

dmz.config:lookup_event_type (name, [default])

Looks up the attribute value name and returns it as a dmz.event_type. If the attribute name is not found, then default is returned. Returns nil if a dmz.event_type can not be found.

dmz.config:lookup_message (name, [default])

Looks up the attribute value name and returns it as a dmz.message. If the attribute name is not found, then default is returned. Returns nil if a dmz.message can not be found.

dmz.config:lookup_object_type (name, [default])

Looks up the attribute value name and returns it as a dmz.object_type. If the attribute name is not found, then default is returned. Returns nil if a dmz.object_type can not be found.

dmz.config:lookup_state (name, [default])

Looks up the attribute value name and returns it as a dmz.mask. If the attribute name is not found, then default is returned.
